As if talking to a non-tech friend --

In the javascript language, describe all of the cases used for each -

Q: parentheses () - e.g. '1. Execute a function var result = doSomething()'

A: These are used for passing parameters (including no parameters) to a function, and also for calling a function in code.

Q: brackets []

A: These square brackets are used to refer to or create an array object, which is essentially a special type of object, zero-indexed, with a number for the name (ie part of each name-value pair).

Q: braces {}

A: Braces are used to define an object, or to define the boundaries of a function.

Q: single quotes '', & double quotes ""

A: These quotes can be used interchangeably in code (as long as they match at both ends eg " " and ' ' only).
