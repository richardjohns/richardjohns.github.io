As if talking to a non-tech friend, discuss:

Q: How does JavaScript compare to HTML and CSS?

Javascript is a programming language that we can use to interact with html and css elements as part of the DOM. It's also used to develop the back-end of a website ie adding, accessing or manipulating information in a related database.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow means the order in which individual statements, instructions, or function calls of an interactive program are executed or evaluated. An example in real life for something like brushing my teeth would involve the individual steps and dependencies that need to occur in order for my teeth to wind up clean. Naturally I'd grab a toothbrush and toothpaste, then I'd need to apply toothpaste to the brush, and brush my teeth! Failing to carry out those steps (or control-flow) would result in my teeth not getting cleaned.

Q: Explain the difference between accessing data from arrays and objects.

An object is a collection of 1 or more name/value pairs. These objects can hold these pairs in the form of an array, being a discrete number of name/value pairs. It's also possible to nest multiple arrays within an object and then again nest arrays within arrays to any degree required.

Q:Explain what functions are and why they are useful.

Functions are useful as at a base level they enable us to take data, perform an operation on the data, and to use the output as needed.
